როგორ გავხსნათ ტერმინალი?

ინსტრუქციაში მოცემული ბრძანებები გაუშვით PowerShell-ში (Windows) ან Terminal-ში (Mac).

🪟 Windows PowerShell (Admin)

  1. კლავიშები Windows ⊞
  2. ჩაწერეთ PowerShell
  3. მარჯვენა კლიკი → Run as administrator
  4. დააჭირეთ Yes

🍎 Mac Terminal

  1. კლავიშები ⌘ + Space
  2. ჩაწერეთ Terminal
  3. დააჭირეთ Enter
💡
ბრძანების გაშვება: დააკოპირეთ ბლოკი (Ctrl+C / ⌘+C) → ჩასვით ტერმინალში (მარჯვენა კლიკი ან ⌘+V) → Enter

Windows-ის პარამეტრების გამართვა 🪟 Windows only
🍎
Mac-ის მომხმარებლებმა ეს ნაბიჯი გამოტოვეთ — გადადით ნაბიჯი 2-ზე.

1.1 — Developer Mode-ის ჩართვა

  1. დააჭირეთ Windows + I (Settings)
  2. ძებნაში ჩაწერეთ Developer settings → დააწკაპუნეთ
  3. Developer Mode → გადართეთ On → Yes

1.2 — ვირტუალიზაციის შემოწმება

  1. დააჭირეთ Ctrl + Shift + Esc (Task Manager)
  2. გადადით Performance → CPU
  3. Virtualization → უნდა წეროს Enabled
⚠️
თუ წერია Disabled — დაუკავშირდით გიორგი ამირიძეს ან სულხან ჯაშს.

1.3 — WSL და Hyper-V

გახსენით PowerShell (Admin) და სათითაოდ გაუშვით:

wsl --install
Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Hyper-V -All
Enable-WindowsOptionalFeature -Online -FeatureName VirtualMachinePlatform -All

🪟 Windows — 1 კლიკით ინსტალაცია რეკომენდებული

ფაილი: install_windows.bat — პატარა, უსაფრთხო ფაილი

👇 ჩამოტვირთვის შემდეგ, ზუსტად მიყევით ამ ნაბიჯებს:

  1. ზემოთ დააჭირეთ ლურჯ ღილაკს — ჩამოიტვირთება ფაილი install_windows.bat.
  2. ბრაუზერმა შეიძლება გაჩვენოთ გაფრთხილება (მაგ. „install_windows.bat is not commonly downloaded"). დააჭირეთ ⋯ (სამი წერტილი) → Keep / შენახვა, შემდეგ Keep anyway. ⚠️ ეს ფაილი უსაფრთხოა.
  3. გახსენით ჩამოტვირთვების საქაღალდე: დააჭირეთ კლავიშებს Ctrl + J (ან ბრაუზერის ზედა მარჯვენა კუთხეში ↓ ისარს) და იპოვეთ install_windows.bat.
  4. ორმაგად დააწკაპუნეთ ფაილზე install_windows.bat.
  5. თუ გამოვა ლურჯი ფანჯარა „Windows protected your PC" — დააჭირეთ More info, შემდეგ Run anyway.
  6. გამოვა ფანჯარა „Do you want to allow this app to make changes?" — დააჭირეთ Yes. (ეს აძლევს ინსტალერს უფლებას დააყენოს პროგრამები.)
  7. გაიხსნება მუქი ფანჯარა და დაგისვამთ კითხვას — აკრიფეთ Y და დააჭირეთ Enter. (ფანჯრის ტექსტი ინგლისურადაა — ეს ნორმალურია.)
  8. ⏳ დაელოდეთ 10–20 წუთი (ინტერნეტის სიჩქარეზეა დამოკიდებული). დაინახავთ, როგორ ყენდება პროგრამები ერთმანეთის მიყოლებით.
  9. ბოლოს დაინახავთ შედეგს (მაგ. „7 / 7"). დააჭირეთ Enter ფანჯრის დასახურად. მზადაა! ✅
💡
თუ ინსტალაციის ბოლოს დაგისვამთ კითხვას გადატვირთვაზე (restart) — აკრიფეთ Y და გადატვირთეთ კომპიუტერი. ეს საჭიროა WSL-ისთვის.

🆘 თუ რამე ვერ გამოვიდა — აქ ნახავთ გამოსავალს

დააჭირეთ პრობლემას, რომელსაც წააწყდით — გაიხსნება გამოსავალი:

🪟 „Windows protected your PC" — ლურჯი ფანჯარა

ეს Windows-ის ჩვეული გაფრთხილებაა. დააჭირეთ More info, შემდეგ გამოჩნდება ღილაკი Run anyway — დააჭირეთ მას. ფაილი უსაფრთხოა.

🪟 ბრაუზერმა ფაილი დაბლოკა / წაშალა („.bat can harm…")

ბრაუზერი ფრთხილობს, რადგან ეს ინსტალაციის ფაილია. ჩამოტვირთვების სიაში იპოვეთ ფაილი, დააჭირეთ ⋯ (სამი წერტილი) ან ისარს გვერდით → Keep / შენახვა → Keep anyway.

🪟 „Yes" ღილაკი ვერ დავაჭირე (გამონაცრისფერებულია / ითხოვს პაროლს)

თქვენს ანგარიშს არ აქვს ადმინისტრატორის უფლება ამ კომპიუტერზე. დაუკავშირდით გიორგი ამირიძეს ან სულხან ჯაშს — ისინი დაგეხმარებიან.

🪟 „running scripts is disabled on this system"

ეს ხდება, თუ სკრიპტს პირდაპირ უშვებთ. გამოსავალი: გამოიყენეთ install_windows.bat ინსტალერი (ზემოთ) — ის ავტომატურად უვლის ამ შეზღუდვას.

🪟 ფანჯარა მაშინვე იხურება / არაფერი ხდება

დარწმუნდით, რომ იყენებთ install_windows.bat ფაილს (და არა პირდაპირ .ps1-ს). თუ .bat-იც არ იხსნება — მარჯვენა კლიკი ფაილზე → Run as administrator. თუ მაინც არ ეშვება, დაუკავშირდით გიორგის ან სულხანს.

🪟 „winget is not recognized" ან პროგრამები ვერ დაყენდა

Microsoft Store-ში მოძებნეთ და დააინსტალირეთ „App Installer", შემდეგ თავიდან გაუშვით ინსტალერი. ან დააინსტალირეთ დარჩენილი პროგრამები ხელით ნაბიჯი 2-დან.

🍎 Mac: „command not found: brew" ინსტალაციის შემდეგ

დახურეთ Terminal სრულად (⌘+Q) და თავიდან გახსენით, შემდეგ ისევ გაუშვით ბრძანება. ეს განაახლებს გარემოს.

🍎 Mac: პაროლი ვერ ვწერ (არაფერი ჩანს)

ეს განზრახაა — Terminal პაროლის აკრეფისას სიმბოლოებს არ აჩვენებს უსაფრთხოებისთვის. უბრალოდ აკრიფეთ თქვენი პაროლი (თითქოს ბრმად) და დააჭირეთ Enter.

🍎 Mac: „xcode-select" ფანჯარა გამოვიდა

დააჭირეთ Install გამოსულ ფანჯარაში და დაელოდეთ დასრულებას. შემდეგ დაბრუნდით Terminal-ში და დააჭირეთ Enter გასაგრძელებლად.

🛡️ ანტივირუსმა/კორპორატიულმა ქსელმა დაბლოკა

ზოგი ანტივირუსი ან სამსახურის ქსელი ბლოკავს ინსტალაციის ფაილებს. სცადეთ სხვა ინტერნეტ ქსელი (მაგ. მობილური), ან დააინსტალირეთ პროგრამები ხელით ნაბიჯი 2-დან. საჭიროების შემთხვევაში დაუკავშირდით გიორგის ან სულხანს.
