Coinmania — AI Training მომზადების ინსტრუქცია

ეს ინსტრუქცია დაგეხმარებათ მოამზადოთ კომპიუტერი აგენტური AI-ის ტრენინგისთვის.
მიყევით ნაბიჯებს ზემოდან ქვემოთ, თანმიმდევრობით.

📌 როგორ გავუშვათ ბრძანებები?

ამ ინსტრუქციაში შეხვდებით მუქ ბლოკებში ჩასმულ ბრძანებებს. ისინი უნდა გაუშვათ PowerShell-ში (Windows) ან Terminal-ში (Mac).

🪟 Windows — PowerShell-ის გახსნა ადმინისტრატორად

  1. კლავიატურაზე დააჭირეთ Windows ⊞ ღილაკს.
  2. ჩაწერეთ PowerShell.
  3. გამოსულ Windows PowerShell-ზე მარჯვენა კლიკი → Run as administrator → Yes.

🍎 Mac — Terminal-ის გახსნა

  1. დააჭირეთ Command ⌘ + Space.
  2. ჩაწერეთ Terminal → Enter.
💡
ბრძანების გაშვება: დააკოპირეთ ბლოკი (Ctrl+C / ⌘+C), ჩასვით PowerShell-ში ან Terminal-ში (მარჯვენა კლიკი / ⌘+V), დააჭირეთ Enter.

⚙️ ნაბიჯი 1: Windows-ის პარამეტრების გამართვა

🍎
Mac-ის მომხმარებლებმა ეს ნაბიჯი გამოტოვეთ — გადადით ნაბიჯი 2-ზე.

1.1 — Developer Mode-ის ჩართვა

  1. დააჭირეთ Windows + I (Settings).
  2. ძებნაში ჩაწერეთ Developer settings → დააწკაპუნეთ შედეგზე.
  3. Developer Mode — გადართეთ On → Yes.

1.2 — ვირტუალიზაციის შემოწმება

  1. დააჭირეთ Ctrl + Shift + Esc (Task Manager).
  2. გადადით Performance → CPU.
  3. Virtualization — უნდა წეროს Enabled.
⚠️
თუ წერია Disabled — დაუკავშირდით გიორგი ამირიძეს ან სულხან ჯაშს.

1.3 — WSL და Hyper-V — გაუშვით სათითაოდ PowerShell (Admin)-ში:

wsl --install
Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Hyper-V -All
Enable-WindowsOptionalFeature -Online -FeatureName VirtualMachinePlatform -All
🔄
თუ გეკითხებათ (Y/N) — ჩაწერეთ Y → Enter. კომპიუტერი გადაიტვირთება.

🛠️ ნაბიჯი 3: CLI ვერსიების დაყენება

გახსენით PowerShell (Admin) ან Terminal და სათითაოდ გაუშვით ბრძანებები.

🟠 Claude Code CLI

Windows — PowerShell (Admin):

irm https://claude.ai/install.ps1 | iex

Mac — Terminal:

curl -fsSL https://claude.ai/install.sh | bash

შემოწმება — გაუშვით ორივე პლატფორმაზე:

claude --version

🟢 Codex CLI (ChatGPT)

Windows:

powershell -ExecutionPolicy ByPass -c "irm https://chatgpt.com/codex/install.ps1 | iex"

Mac:

curl -fsSL https://chatgpt.com/codex/install.sh | sh
ℹ️
პირველი გაშვებისას (codex) გეკითხებათ ავტორიზაცია — აირჩიეთ Sign in with ChatGPT.
